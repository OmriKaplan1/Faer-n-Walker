Project Overview

Faerûn Wanderer combines GPS tracking, gamification, and D&D fantasy elements to create an immersive location-based adventure game that encourages physical activity and exploration.

Real-World GPS Tracking

Uses actual GPS coordinates to track movement and convert walking distance into game steps and experience points.

D&D Character System

Choose from 12 classic D&D classes, level up through walking, and build your character's stats and abilities.

Interactive Map

Explore real-world locations mapped to fantasy settings with points of interest, encounters, and rewards.

Key Features

Step Tracking & Leveling

GPS-based movement detection converts real walking into game steps. Every 1000 steps = 1 level up + gold rewards.

Economy & Shop System

Earn gold through activities and spend it on equipment, weapons, and consumables in the in-game shop.

Monster Encounters

Random monster encounters spawn on the map. Fight creatures appropriate to your level for gold and experience.

Achievement System

Unlock achievements for reaching step goals, visiting locations, and completing challenges.

Technology Stack

Built with modern web technologies and Microsoft Azure cloud services

React Frontend

Built on Base44 platform with React, Tailwind CSS, and Leaflet for interactive maps

Azure Functions

Serverless Python functions for game logic, user management, and data processing

Azure SQL

Relational database for user data, game state, equipment, and location tracking

Azure Maps

High-quality satellite imagery and mapping services for the game world

  • • Equipment icons and images
  • • Static game content storage
  • • CDN integration for fast loading
