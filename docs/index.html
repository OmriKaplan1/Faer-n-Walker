Logic

Our frontend runs on Base44 (rendering, routing, hosting). The client interacts with Base44, and Base44 calls our Azure Functions via the ‚ÄúBackend Functions‚Äù integration. Functions hold the business logic and secrets, reading/writing data in Azure SQL Database and storing media/documents in Azure Blob Storage. Map tiles, geocoding, and visualization use Azure Maps directly from the frontend. This keeps the UI fast, scales compute independently, and centralizes sensitive logic in the cloud.
